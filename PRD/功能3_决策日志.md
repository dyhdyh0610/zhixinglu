# 功能3:决策日志(核心功能)

> 摘自《知行录 产品需求文档》v1.1 第6.3节

---

## 1. 功能定位

**这是整个产品最重要的功能,是产品的"脊椎"**。它的设计好坏,决定了产品的天花板。

## 2. 用户故事

> 作为一个想认真投资的散户,我希望每一次交易决策都能被记录下来——我当时的判断、我的预期、我的情绪——这样未来回看时,我能看到自己的判断准不准,情绪管控好不好,从而真正成长。

## 3. 设计原则

- **极轻**:30秒内必须能完成填写,否则用户会放弃
- **结构化但不死板**:多用选项,少用文本输入
- **价值前置**:用户填写时就要感受到价值,而不是"先填了再说"

## 4. 入口设计

- 底部Tab"我的"中,显著的"+ 记录决策"按钮
- Agent对话中可触发"记录这笔决策"
- 持仓列表中,某只股票上可触发"为这笔交易补记日志"

## 5. 决策卡片填写流程(分3步)

### 第1步:基本信息(目标10秒完成)

**字段**:
| 字段 | 类型 | 输入方式 | 必填 |
|------|------|---------|------|
| 操作类型 | 单选 | 4个大按钮 | 是 |
| 标的 | 搜索框 | 代码或名称搜索 | 是 |
| 价格 | 数字 | 数字键盘 | 是 |
| 数量 | 数字 | 数字键盘 | 是 |
| 占组合比例 | 数字 | 自动计算或用户填写 | 否 |
| 交易日期 | 日期选择 | 默认今天 | 是 |

**操作类型4选**:
- 买入(首次)
- 加仓
- 减仓
- 卖出(清仓)

### 第2步:决策逻辑(目标15秒完成)

**字段**:
| 字段 | 类型 | 说明 |
|------|------|------|
| 决策原因 | 多选 | 至少选1项 |
| 核心假设 | 单行文本(引导式) | 限80字,必填 |
| 预期持有时长 | 单选 | 5个区间 |

**决策原因选项(多选)**:
- ☐ 基本面看好
- ☐ 基本面变差(适用于卖出)
- ☐ 技术面信号
- ☐ 估值便宜/合理
- ☐ 估值过高(适用于卖出)
- ☐ 行业事件/政策利好
- ☐ 行业事件/政策利空
- ☐ 公司事件(财报、并购等)
- ☐ 跟随大V/朋友
- ☐ 直觉/感觉
- ☐ 其他(展开文本框,限50字)

**核心假设(关键字段)**:
- 引导文案:"用一句话说,你的核心判断是______"
- 示例提示(用户可参考):
  - "我认为这家公司Q3营收增速会回升到30%以上"
  - "我认为新能源车销量数据会超预期"
  - "我认为这次减仓后行业还会下跌至少20%"
- 字数限制:80字

**预期持有时长**:
- ○ 1周内(短线)
- ○ 1个月内
- ○ 3个月内
- ○ 半年内
- ○ 1年以上(长期)

### 第3步:风险和情绪(目标5秒完成)

**字段**:
| 字段 | 类型 | 说明 |
|------|------|------|
| 止损位 | 数字(可选) | 触发卖出的价格 |
| 止盈位 | 数字(可选) | 触发卖出的价格 |
| 此刻情绪 | 1-5滑块 | "焦虑"到"平静" |
| 备注 | 文本框(可选) | 限200字 |

**情绪滑块设计**:
- 1分:非常焦虑/恐慌
- 2分:有些紧张
- 3分:平静
- 4分:比较自信
- 5分:非常自信/兴奋

## 6. 完成后的即时反馈

**AI生成"决策画像反馈"**(不评判对错):
- 频率提示:"这是你这个月的第8笔交易,频率比上月增加了60%"
- 假设跟踪:"你这次的核心假设是关于'Q3营收增速',这个假设可以在11月财报时验证。我们会在那时提醒你回顾。"
- 模式识别:"你最近3次买入时情绪都在4-5分(高位),这种状态下你的历史胜率是X%。" (需用户数据积累后启用)

**示例完整反馈**:
> 已记录你这笔关于宁德时代的加仓决策。
> - 这是你这个月的第3笔交易,频率正常
> - 你的核心假设可在Q3财报(预计10月底)验证,我们会到时提醒你
> - 你目前情绪打分是4分,你过往在4分以上做的决策有62%最终是赚钱的
> 时间会告诉你答案。

## 7. 决策日志展示页

### 列表视图

- 按时间倒序排列,每条决策一张卡片
- **卡片正面信息**:
  - 操作类型 + 标的(图标+名称)
  - 操作日期
  - 当时的核心假设(突出显示)
  - 当时的情绪(图标)
- **卡片右上角实时回填**:
  - 距今天数:"42天前"
  - 当前盈亏:"+12.5%"(用低饱和度颜色)
- 点击展开:完整决策卡片 + AI对这笔决策的跟踪状态

### 统计视图

**核心图表**:
1. **交易频率趋势图**:每月交易次数,过去6个月
2. **预期vs实际持有时长**:每笔决策的"预期"和"实际"对比(揭示"做短线变长线套牢"的问题)
3. **情绪vs盈亏散点图**:每笔决策的情绪分和最终盈亏的相关性(揭示情绪化交易模式)
4. **核心假设验证统计**:已到期的假设中,被验证的比例
5. **决策原因分布**:你的决策最常基于什么(基本面/技术面/跟随等)

## 8. 数据所有权

- 用户可随时**导出**所有决策日志(CSV/JSON格式)
- 用户可随时**删除**单条或全部日志
- 平台不会未经用户许可使用日志数据
